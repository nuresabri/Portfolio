Projects

Here are some of my notable projects:

Game Review and Recommendation Platform

This project is a web application built with PHP, the platform features a user authentication system, game reviews, and recommendations, with an admin page for managing users. Users can register, log in, submit game reviews or recommendations, and interact with a list of video game characters. The admin page allows for CRUD operations on user accounts. The application uses SQL to manage data relationships, ensuring seamless updates and deletions. It also requires MAMP for server functionality and PHP support.

AI-Powered Weather App

I developed a weather application using React.js, HTML, and CSS, integrated with an AI-powered weather API. The app provides real-time weather data, including current conditions and forecasts, with a user-friendly interface. I focused on efficient API integration, responsive design, and delivering an intuitive user experience.

GitHub Repository

Web Scraping Game Reviews

A project to scrape reviews of video games using Beautiful Soup and Scrapy. It targets reviews from various eCommerce sites, cleans the data, and saves it in text files for analysis.

GitHub Repository

Enhancing Quicksort

A project that compares different variations of the Quicksort algorithm, including strategies like choosing the pivot from the middle, random pivot, and the default approach. Implemented in multiple languages for performance analysis.
